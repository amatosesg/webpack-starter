# Webpack Starter Template

This is a project to create apps with webpack.

### Notes: 
Remember to install node_modules:
```
npm install
```
To build, you have 2 options.
For production:
```
npm run build
```
or for development:
```
npm run build:dev
```
To run server (default port: 8080):
```
npm run serve
```
